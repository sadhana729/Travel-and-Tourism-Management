
# ✈️ Travel & Tourism Management System

A Java-based desktop application that simplifies travel package management, customer registration, and booking operations through an interactive graphical user interface (GUI). The system demonstrates object-oriented programming concepts while providing an easy-to-use travel management platform.

---

## 📌 Features

- 👤 Customer Management
  - Add new customers
  - View customer details
  - Manage customer records

- 🌍 Travel Package Management
  - Domestic travel packages
  - International travel packages
  - Package details and pricing

- 📝 Booking Management
  - Create bookings
  - Search bookings
  - Display booking details

- 🖥️ Interactive GUI
  - Java Swing based interface
  - User-friendly navigation
  - Dashboard-style layout

- 📂 File Handling
  - Save booking information
  - Read booking records from files

- 🧵 Multithreading
  - Booking processing thread
  - Notification thread

- ⚠️ Exception Handling
  - Custom exceptions
  - Error validation

## 🚀 How to Run

### Clone the Repository

```bash
git clone https://github.com/your-username/travel-tourism-system.git
```

### Navigate to the Project

```bash
cd travel-tourism-system
```

### Compile

```bash
javac *.java models/*.java services/*.java interfaces/*.java exceptions/*.java threads/*.java
```

### Run GUI

```bash
java TravelGui
```

or

```bash
java TravelTourismManagementSystem
```
